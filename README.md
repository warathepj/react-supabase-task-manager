# Vite Supabase Starter

A minimal starter template for building React applications with Vite, TypeScript, and Supabase integration.

## Features

- ⚡️ [Vite](https://vitejs.dev/) - Lightning fast frontend tooling
- ⚛️ [React 19](https://react.dev/) - JavaScript library for user interfaces
- 📘 [TypeScript](https://www.typescriptlang.org/) - Type safety and enhanced developer experience
- 🔋 [Supabase](https://supabase.com/) - Open source Firebase alternative
- 🔍 [ESLint](https://eslint.org/) - Code linting with React-specific rules

## Prerequisites

- Node.js (version 18.0.0 or higher)
- npm (version 9.0.0 or higher)
- A Supabase account and project

## Getting Started

1. Clone the repository:

```bash
git clone <your-repo-url>
cd vite-supabase-starter
```

2. Install dependencies:

```bash
npm install
```

3. Create a `.env.local` file in the project root and add your Supabase credentials:

```env
VITE_SUPABASE_URL=your-supabase-project-url
VITE_SUPABASE_ANON_KEY=your-supabase-anon-key
```

4. Set up your Supabase database:

```sql
-- Create the test table

alter table public.test enable row level security;

-- Create policies
create policy "Enable read access for all users"
on public.test
```
